## ⚙️ 环境变量配置

在青龙面板「环境变量」中添加以下变量：

| 变量名 | 说明 | 必填 |
|--------|------|------|
| `ZWSOFT_USERNAME` | 中望社区账号（手机号/邮箱），多账号换行分隔 | ✅ |
| `ZWSOFT_PASSWORD` | 中望社区密码，多账号换行分隔（与账号一一对应） | ✅ |
| `ZWSOFT_NOTIFY` | 通知级别：0=关闭 1=仅异常 2=全部通知（默认1） | ❌ |
| `ZWSOFT_DEBUG` | 调试模式：true/false（默认false） | ❌ |
| `ZWSOFT_RETRY_COUNT` | 失败重试次数（默认1次，即总共尝试2次） | ❌ |
| `ZWSOFT_RETRY_INTERVAL` | 重试间隔秒数（默认180秒=3分钟） | ❌ |
| `ZWSOFT_QL_NOTIFY` | 是否启用青龙面板通知：true/false（默认：配置WXPusher后自动禁用） | ❌ |
| `WXPUSHER_APP_TOKEN` | WXPusher 应用 Token（微信推送，可选） | ❌ |
| `WXPUSHER_UIDS` | WXPusher 全局接收者 UID，多个用逗号分隔（可选） | ❌ |
| `WXPUSHER_USER_MAP` | 账号与UID映射，不同账号推送到不同微信（可选） | ❌ |

### 多账号配置示例

**ZWSOFT_USERNAME:**
```
13800138000
13900139000
```

**ZWSOFT_PASSWORD:**
```
password123

> 💡 账号和密码按顺序一一对应，第1行账号对应第1行密码

```

### 方式二：WXPusher 微信推送（推荐）

如果不想配置青龙全局通知，也可以直接通过环境变量配置 WXPusher：

**配置步骤：**

1. 访问 [WXPusher 官网](https://wxpusher.zjiecode.com/) 注册账号
2. 创建应用，获取 `APP_TOKEN`
3. 关注应用的公众号，获取你的 `UID`
4. 在青龙面板环境变量中添加：

| 变量名 | 说明 | 示例 |
|--------|------|------|
| `WXPUSHER_APP_TOKEN` | 应用 Token | `AT_xxx...` |
| `WXPUSHER_UIDS` | 接收者 UID，多个用逗号分隔 | `UID_xxx,UID_yyy` |

**通知效果：**
```
✅ 签到成功！
   账号：138****4746
   连续签到: 41 天
   获得积分: 31
   总积分: 508
```

> 💡 **提示**：配置 WXPusher 后，脚本会**自动禁用青龙面板通知**，避免同时收到两条相同的推送消息。
>
> 如需同时启用两种通知方式，可添加环境变量 `ZWSOFT_QL_NOTIFY=true`。

### 方式三：多账号分别推送到不同微信

如果有多个账号，想推送到不同的微信，可以使用 `WXPUSHER_USER_MAP` 配置：

**配置格式：**
```
WXPUSHER_USER_MAP = 账号1=UID1,账号2=UID2,账号3=UID3
```

**支持的分隔符：**
- 账号之间：逗号 `,`、分号 `;`、换行
- 账号与 UID 之间：等号 `=`、冒号 `:`

**示例：**
```
WXPUSHER_USER_MAP = 18103866761@163.com=UID_张三,138xxxx4746=UID_李四
```

**智能合并：**
- 如果多个账号映射到同一个 UID（同一个微信），会**自动合并成一条消息**
- 不会重复推送，一条消息包含所有该微信的账号签到结果

**混合配置：**
- 配置了 `WXPUSHER_USER_MAP` + `WXPUSHER_UIDS`
- 在映射中的账号 → 推送到映射的 UID
- 不在映射中的账号 → 推送到全局 `WXPUSHER_UIDS`

### 通知级别控制

通过 `ZWSOFT_NOTIFY` 环境变量控制：

- `0`：关闭所有通知
- `1`（默认）：仅异常时通知（签到失败、脚本错误）
- `2`：每次运行都通知

## 🔧 常见问题

### Q: 签到失败怎么办？

A: 脚本默认会自动重试 1 次（间隔 3 分钟）。可以通过环境变量调整重试次数和间隔：
```
ZWSOFT_RETRY_COUNT = 2      # 重试2次
ZWSOFT_RETRY_INTERVAL = 300  # 间隔5分钟
```

### Q: 多账号怎么配置？

A: 在环境变量中每行一个账号，密码同理，顺序一一对应。

### Q: 收不到通知？

A: 1. 检查 WXPusher 的 APP_TOKEN 和 UID 是否正确
   2. 确认 `ZWSOFT_NOTIFY` 设置为 2（成功也通知）
   3. 开启 `ZWSOFT_DEBUG=true` 查看详细调试日志

### Q: 不同账号怎么推送到不同微信？

A: 使用 `WXPUSHER_USER_MAP` 环境变量配置映射关系：
```
WXPUSHER_USER_MAP = 账号A=UID_A,账号B=UID_B
```

### Q: 依赖怎么安装？

A: 青龙面板中在「依赖管理」→「Python3」中添加：
- `requests`
- `pycryptodome`
